Lesson 1: Variables, Data Types, and Output

Code Snippet: Declaring Variables and Outputting Data


let userName = "Alex";
const year = 2025;

console.log("Hello, " + userName + "! The year is " + year + ".");
document.getElementById("output").textContent = `Welcome, ${userName}!`;

Step-by-Step Explanation

  1. let creates a variable whose value can change.
  2. const creates a constant whose value cannot change.
  3. console.log() prints a message to the browser's console (press F12 to view).
  4. document.getElementById().textContent outputs a message to an element in your HTML.

Exercise

  • Declare a variable for your favorite color and print it using console.log.
  • Add a <div id="js-output"></div> to your HTML and display a message using JavaScript.

Lesson 2: Functions and Events

Code Snippet: Click Event Handler


document.getElementById("myButton").onclick = function() {
  alert("Button was clicked!");
};

Explanation

  1. Selects the button with id myButton.
  2. Assigns a function to its onclick event.
  3. Shows an alert when the button is clicked.

Exercise

  • Create a button that changes the background color of the page to a random color when clicked.

Lesson 3: Arrays, Loops, and DOM Manipulation

Code Snippet: Displaying a List


const fruits = ["Apple", "Banana", "Cherry"];
const ul = document.createElement("ul");

for(let fruit of fruits) {
  const li = document.createElement("li");
  li.textContent = fruit;
  ul.appendChild(li);
}
document.body.appendChild(ul);

Explanation

  1. Defines an array fruits.
  2. Creates a <ul> element.
  3. Loops through the array, creates <li> for each item, and appends to the list.
  4. Appends the list to the page.

Exercise

  • Make a JavaScript array of your three favorite programming languages and display them as a list on the page.
